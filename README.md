# KANG-ZHAOYUAN.github.io

## 项目简介

这个仓库是我用来练习创建网站的项目。它没有其他特殊的用途，主要作为个人学习和实验的平台。通过这个仓库，我在不断尝试和提升网页开发技能。

## 项目目的

- 学习 HTML、CSS、JavaScript 等前端技术。
- 熟悉 Git 和 GitHub 的使用。
- 练习如何通过 GitHub Pages 部署静态网站。

## 项目结构

- **index.html**: 网站的主页。
- **style.css**: 样式文件。
- **scripts.js**: 页面交互的 JavaScript 脚本。
