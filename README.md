## 项目简介

这是一个伪装成“今天日期”的小站，背后其实藏着一个为 **“茜特菈莉” 发癫** 的玩具级加密/解密器（演示用，纯娱乐）。表面上我在练习前端——实际上是为了让茜特菈莉的秘密看起来更神秘。🔐🎭

## 项目目的

- 用最俏皮的方式练手 HTML、JavaScript 与小工具开发。🎨
- 演示一种将外部数据（例如“今天的日期”）作为小道具来驱动加密想法的方式。🗝️
- 提供一个容易扩展的基础：想把它变成真正的加解密小玩具？改造 `get_date.js` 就能开始。🛠️

## 项目结构（快速参考）

- **index.html**: 页面入口，显示“今天的日期”（以及一条编码提示小字）。
- **get_date.js**: 获取日期、自动刷新、失败回退的主要逻辑；也是你改造加密/解密逻辑的起点。💻
- **README.md**: 你正在阅读的俏皮使用说明。😉

## 使用说明

1. 克隆仓库：
   ```bash
   git clone git@github.com:KANG-ZHAOYUAN/KANG-ZHAOYUAN.github.io.git
   ```
2. 在本地打开 `index.html` 或运行：
   ```bash
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```
3. 想要把它变成真正的“茜特菈莉”加密器？打开 `get_date.js`，写几个 `encrypt()` / `decrypt()` 函数，把“日期”作为灵感或密钥的一部分，尽情发挥。🧪

## 有趣的“彩蛋”说明 ✨

- 表面功能：显示“今天是：YYYY年MM月DD日”，并每 60 秒自动刷新（可调整）。
- 失败处理：当外部请求失败时，页面会优雅地回退为本地时间并停止自动刷新，显示提示信息以保护 API 配额。😌
- 你可以把“今天的日期”当作“茜特菈莉”密语的起点，但本仓库默认只演示流程，不负责任何严肃安全用途。🛡️💬

## 配置与注意事项

- 请在 `get_date.js` 顶部填入你的 News API Key（`api_key`）。
- 注意 API 配额：别太猛刷新（默认 60 秒），否则可能触发限流。⏱️
- 若看到中文乱码请确认文件以 **UTF‑8（无 BOM）** 保存（详见下文）。

---

## 故障排查 — 编码问题 🔧

- 中文显示异常时，请在编辑器中将 `index.html`、`get_date.js` 等文件保存为 **UTF‑8（无 BOM）**。
- 在 VS Code：右下角编码处点击 → 选择 “Reopen with Encoding” 验证内容 → 然后 “Save with Encoding” 选择 `UTF-8`。
